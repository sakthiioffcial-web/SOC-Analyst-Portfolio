# Day 3 – Linux Fundamentals Practice

## Commands Practiced
- pwd → show current directory
- ls → list files
- cd .. → move up
- mkdir test → create folder
- touch file1.txt → create file
- echo "Hello World" > file1.txt → write to file
- cat file1.txt → read file
- cp file1.txt copy.txt → copy file
- mv copy.txt moved.txt → rename file
- rm moved.txt → delete file
- ls -l → list with permissions
- chmod 644 file1.txt → change permissions
- whoami → current user

## Log Analysis

- Basic Linux navigation and file operations
